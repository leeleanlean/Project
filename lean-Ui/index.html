代码约定

移动端需要添加特殊的meta标签

<meta name="viewport" content="initial-scale=1.0,width=device-width,user-scalable=no" />
页面放大到设备实际尺寸 N 倍，一般设置1.0，禁止用户缩放页面

<meta content=“email=no” name=“format-detection” />
不要将页面中的数字识别为电话号码 1、代码片段出现的 .className 均为 Submit Text编辑器的emmet 快捷键 2、代码片段中出现的"*n" n代表需要添加的数目 3、代码片段中出现的"&&" 表示并且的意思，需要两个class同时具备 4、多个效果只需添加多个Class Example：class="box-shadow scale-x m-t-10"
